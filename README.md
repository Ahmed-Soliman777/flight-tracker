# ✈️ Flight Tracker

## About the Project

**Flight Tracker** is a modern web application that provides real-time aircraft tracking through an interactive map interface. The application allows users to explore live flights around the world, view detailed flight information, and monitor aircraft movements in an intuitive and responsive user experience.

The project focuses on building a clean, performant, and scalable frontend while integrating real-time aviation data from external APIs. It also demonstrates working with interactive maps, asynchronous data fetching, and state management in a modern React ecosystem. The implementation is inspired by a real-time flight tracking tutorial using Next.js, Tailwind CSS, React Map GL, and the OpenSky Network API.

---

## Technologies

The project will be built using the following technologies:

* **Next.js**
* **React**
* **TypeScript**
* **Tailwind CSS**
* **React Map GL**
* **Mapbox**
* **OpenSky Network API**
* **Fetch API**

---

## Features

The application will include the following features:

* 🌍 Interactive world map for displaying live aircraft positions.
* ✈️ Real-time flight tracking.
* 📍 Dynamic aircraft markers with heading rotation based on flight direction.
* 🎯 Click on any aircraft to view detailed flight information.
* 🛫 Display departure and arrival airport information.
* 🗺️ Show the selected aircraft's flight trajectory.
* 🔍 Search for flights or aircraft.
* 📱 Fully responsive design for desktop, tablet, and mobile devices.
* ⚡ Fast and optimized performance using Next.js.
* 🎨 Modern and clean user interface.
* ✨ Smooth animations and transitions using Framer Motion.
* 🔄 Automatic data refresh to keep flight information up to date.
* 🌐 Interactive map controls including zooming, panning, and navigation.

## Project Goals

This project aims to:

* Build a real-world application using modern frontend technologies.
* Practice working with external REST APIs.
* Learn how to integrate interactive maps into React applications.
* Improve state management across multiple components.
* Create a scalable and maintainable project structure.
* Deliver a responsive and user-friendly experience for visualizing real-time aviation data.
